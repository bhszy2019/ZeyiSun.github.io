### 1. 基本模式

matlab以矩阵作为单位，每个变量都以一个矩阵的形式存储，绘图实际上是一个打点连线的过程。当打的点很多时,绘图的结果从外表上看来就是变成了光滑的曲线，以下面所示绘图为例（x为1*41的矩阵）：

```matlab
x = [-2*pi : 0.1*pi : 2*pi]; % x为一个1*41的矩阵（也可以视作一个行向量）
y = sin(x);                  % 此行执行后，y将作为一个与x同形的矩阵存储，且yij = sin(xij)(for i,j in x矩阵)
plot(x, y)                   %此步系统调用将进行打点绘图
```

![](https://github.com/SunzeY/SunzeY.github.io/_posts/MM[01]/1.jpg)

调整x的步长， 打点的精度也会随之改变。三维绘图实际上是空间中打点并连点成线（三维曲线）或者连点成网（三维曲面），与此同理。



当然，绘图过程往往涉及到矩阵的点积，而非一般意义上的矩阵乘法。我们必须用 ".*" 来表示点积，示例如下：

```matlab
x = [-10: 0.1: 10];
m = exp(0.1*x)
y = m .* sin(x);   % m与sin(x)两个列向量进行内积的结果赋值给y，y与x作为矩阵的形状和大小完全相同
plot(x, y)
```

![](https://github.com/SunzeY/SunzeY.github.io/_posts/MM[01]/2.jpg)



当然，三维绘图往往涉及到矩阵的点积，而非一般意义上的矩阵乘法。我们必须用 ".*" 来表示点积，示例如下：

```matlab
a = linspace(-10, 10, 100);
[x, y] = meshgrid(a); % 将在后面介绍此处发生了什么
z = x.^2 + y.^2;      % 这里等价于 z = x.*x + y.*y
surf(x, y, z);
```

![]((https://github.com/SunzeY/SunzeY.github.io/_posts/MM[01]/3.jpg)

x.*x 的结果是[xij * xij  for xij in x]，其结果是x矩阵自己进行内积（这里的x实际上为100*100的矩阵）。
